# Customer Churn & Retention Analysis Dashboard

## 📌 Project Overview

This project analyzes customer churn behavior and retention trends using SQL and Power BI.

## 🛠 Tools & Technologies

* SQL
* Power BI
* Excel

## 📊 Key Features

* Churn Rate & Retention Rate analysis
* Customer segmentation by subscription type
* Identification of high-risk customers
* Interactive dashboard with filters and KPIs

## 🔍 Key Insights

* Customers with lower tenure show higher churn
* Basic subscription plan has higher churn rate
* High-risk customers identified for proactive retention strategies

## 📁 Files Included

* Power BI Dashboard (.pbix)
* SQL Queries
* Dataset (sample)

## 🚀 Outcome

Enabled data-driven decision-making by identifying churn patterns and improving customer retention strategies.
